# Avances por etapa

Este directorio contiene un archivo por cada paso del plan de implementación.

## Convención usada
- Un archivo por etapa: `paso-01.md` ... `paso-12.md`.
- Cada archivo incluye:
  - checklist de indicadores del paso,
  - grado de cumplimiento por indicador,
  - estado consolidado (porcentaje + semáforo + observación).
