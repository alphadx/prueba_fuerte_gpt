# Skills para implementación ERP (pasos 1 a 12)

Estas skills están orientadas a trabajo modular, arquitectura, optimización y confiabilidad para el plan ERP.

## Mapeo paso -> skill
- **Paso 1:** `mvp-scope-qa-gate`
- **Paso 2:** `modular-monorepo-architecture`
- **Paso 3:** `docker-compose-core-stack`
- **Paso 4:** `data-model-migrations-blueprint`
- **Paso 5:** `api-authz-modular-gateway`
- **Paso 6:** `pos-cashflow-reliability`
- **Paso 7:** `billing-sandbox-adapter`
- **Paso 8:** `payment-gateway-idempotency`
- **Paso 9:** `ecommerce-pickup-orchestration`
- **Paso 10:** `hr-docs-alerts-engine`
- **Paso 11:** `test-state-bootstrap-factory`
- **Paso 12:** `release-readiness-observability-gates`

## Enfoque complementario
Cada skill incluye:
- workflow operativo,
- buenas prácticas y malas prácticas,
- tendencias actuales,
- enfoque de reliability,
- alineación con aspiraciones de mercado (KPIs y outcomes esperados).

## Objetivo
Estandarizar cómo se ejecuta cada etapa, con workflows reutilizables y checklists de cierre para reducir ambigüedad, aumentar velocidad de implementación y mejorar competitividad técnica.
